##  Sprint Goal
Establish a rock-solid foundation by installing professional tools (Python, VS Code, Docker) and gaining the ability to write scripts that can process, filter, and transform bank transaction data.

---

##  Sprint Backlog

###  Level 0 — Environment & Tooling (The Foundation)
- [ ] **Python Installation**
- [ ] **VS Code Setup:** Install Visual Studio Code with essential extensions (Python, Pylance, Docker, GitLens).
- [ ] **Git Configuration:** Set up the local repository and connect it to GitHub.
- [ ] **Virtual Environments:** Master `venv` to isolate project dependencies.
- [ ] **Docker Basics:** Install Docker Desktop and run the `hello-world` container.

###  Level 1 — Logic Foundations (Core Basics)
- [ ] **Primitive Data Types:** Working with `Integers`, `Strings`, `Floats`, and `Booleans`.
- [ ] **Complex Data Structures:** Mastering `Lists` , `Dictionaries` (JSON-like structures), and other structures.
- [ ] **Control Flow:** Implementing business logic with `if/elif/else` and automation with `for` and `while` loops.
- [ ] **Built-in Functions:** Using `len()`, `sum()`, `range()`, `enumerate()`, `zip()` , and others .

###  Level 2 — Code Organization (Software Engineering)
- [ ] **Functions:** Writing reusable logic with `def`, arguments, and `return` statements.
- [ ] **Type Hinting:** Securing code readability (e.g., `def clean(data: dict) -> str:`).
- [ ] **Modularity:** Organizing code into multiple `.py` modules.
- [ ] **Error Handling:** Mastering `try/except` blocks to prevent pipeline crashes.

###  Level 3 — Data Specialization (The Data Engineer's Edge)
- [ ] **File Manipulation:** Reading and writing `CSV` and `JSON` files.
- [ ] **Datetime Management:** Handling timestamps and timezones for banking records.
- [ ] **Logging:** Implementing professional logging instead of simple `print()` statements.
- [ ] **Pandas Introduction:** Creating and inspecting DataFrames (The Golden Tool).
